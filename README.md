# テキストファイルで見出しを作る方法

このドキュメントでは、テキストファイルで見出しを作成する方法を説明します。

## Markdown形式の見出し

テキストファイルで見出しを作成するには、`#`（シャープ）記号を使用します。これはMarkdown形式と呼ばれる記法です。

### 見出しのレベル

見出しには6つのレベルがあります：

```
# 見出し1（最も大きい）
## 見出し2
### 見出し3
#### 見出し4
##### 見出し5
###### 見出し6（最も小さい）
```

### 使用例

```
# メインタイトル
本文が始まります。

## セクション1
セクション1の内容です。

### サブセクション1-1
詳細な内容です。

## セクション2
セクション2の内容です。
```

### 重要なポイント

1. **`#`の後にスペースを入れる**: `# 見出し` のように、`#`の後に半角スペースを入れてください
2. **見出しの前後に空行を入れる**: 見出しの前後に空行を入れると読みやすくなります
3. **適切なレベルを使う**: 
   - `#` は文書のメインタイトルに使用（通常は1つだけ）
   - `##` は主要なセクションに使用
   - `###` 以下はサブセクションに使用

### ファイル拡張子

- `.txt` ファイル: Markdown記法を使えますが、表示時は通常のテキストとして表示されます
- `.md` ファイル: Markdownファイルとして認識され、GitHubなどで自動的に整形表示されます

## その他の整形方法

### 箇条書き

```
- 項目1
- 項目2
  - サブ項目2-1
  - サブ項目2-2
```

### 番号付きリスト

```
1. 最初の項目
2. 2番目の項目
3. 3番目の項目
```

### 強調

```
**太字**
*斜体*
```
